# Beach Holiday Park Management System
**Comprehensive Object-Oriented System Design (UML & OOAD)**

### Project Summary
This repository contains a full-scale architectural design for a Beach Holiday Park. The system manages complex dependencies between **Facility Management** (Restaurant, Cleaning, Supplies) and **Booking Operations** (User Reservations, Scheduling).

### Design Artifacts (Summary)
The full documentation includes **40+ technical artifacts**, demonstrating a complete SDLC design phase:
* **20+ UML Diagrams:** Including Activity, Class, Sequence, State Machine, and Interactive Overview diagrams.
* **10 Use Case Specifications:** Detailed behavioral requirements for all system actors.

### Key Design Highlight: Multi-Manager Supply Logic
I designed a centralized **Supply Management (UC900)** system that ensures data integrity across multiple departments (Cleaning, F&B, Facilities). 

**Trust & Safety Impact:** * **Role-Based Access Control:** Strict Actor definitions (A100-Manager) prevent unauthorized data modification.
* **Scalability:** Categorized inventory structures allow the system to scale as the park grows without data collision.
